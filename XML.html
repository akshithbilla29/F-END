XML(Extensible Markup Language)

What is XML?

  • XML (Extensible Markup Language) is a markup language that is designed to store, organize and transport data in a human-readable and machine-readable format. It is a simple and flexible text-based format that is widely used in various domains including web services, software development, data exchange and storage, and electronic publishing.
  • XML consists of a set of rules for encoding documents in a format that can be easily read by both humans and machines. In XML, data is structured into elements, which are defined by tags. Each element can have attributes that provide additional information about the element.
